= Monitorar trabalhos de cache de volume na página Auditoria
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode monitorar todos os trabalhos de cache de volume e determinar seu progresso usando a página Auditoria do NetApp Console.

.Passos
. No menu do NetApp Console, selecione *Administração* > *Auditoria*.
. Opcionalmente, filtre por tempo, serviço, ação, agente, recurso, usuário ou status.
. Explore todos os caches e as operações.
