Você pode usar o cache de volume do NetApp para acelerar o acesso aos dados ou para descarregar o tráfego de volumes muito acessados. Os volumes de cache são ideais para cargas de trabalho com uso intensivo de leitura, especialmente quando os clientes precisam acessar os mesmos dados repetidamente.

= Monitore os trabalhos de cache de volume na página de auditoria.
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode monitorar todos os trabalhos de cache de volume e determinar seu progresso usando a página de auditoria do NetApp Console .

.Passos
. No menu do NetApp Console , selecione *Administração* > *Auditoria*.
. Opcionalmente, filtre por tempo, serviço, ação, agente, recurso, usuário ou status.
. Explore todos os caches e as operações.
